Project 1: Student Management System

Overview: A comprehensive student management system designed for educational institutions. It streamlines registration, data management, and statistical analysis.

  • Key Features:
  • Student Registration: Add, remove, and update student records.
  • Dynamic Statistics Panel: Compare courses and enrollment years with charts.
  • Data Visualization: Intuitive bar and pie charts using JFreeChart.
  • Database Integration: Secure storage with MySQL.

Technologies Used: Java (Swing), MySQL, JFreeChart, Maven.

Learning Outcomes: Mastered pagination, chart creation, and database management.

Next Steps: Role-based access control, multi-language support, and web deployment.

Project 3: Cafeteria Ordering App

Overview: A mobile app built to simplify cafeteria orders, minimize waiting times, and enhance the ordering experience for students.

  • Key Features:
  • Place Orders: Add food items to the cart with real-time updates.
  • Cart Management: Swipe left to delete items.
  • Order Status: Pop-up timer notifies when orders are ready.
  • Recent Orders: Tracks past orders with their respective dates.
  • Firebase Integration: Stores user details, order history, and authentication data securely.

Technologies Used: Kotlin, Android Studio, Firebase.

Learning Outcomes: Enhanced skills in Firebase authentication and database integration, swipe gestures, and dynamic UI updates.

Next Steps: Add payment gateway integration and a notification system for special offers.

Project 4: Mobile Finance App

Overview: A Kotlin-based finance management app that provides users with essential tools to manage their finances effectively.

  • Key Features:
  • Account Management: Register and log in securely.
  • Transactions: Deposit, withdraw, transfer funds, and verify account numbers.
  • Transaction History: View detailed logs of all past transactions.
  • Balance Inquiry: Check account balance instantly.

Technologies Used: Kotlin, IntelliJ IDEA, SharedPreferences.

Learning Outcomes: Improved proficiency in persistent data storage, secure user authentication, and object-oriented design in Kotlin.

Next Steps: Include data encryption for added security and integrate cloud storage for seamless multi-device support.
